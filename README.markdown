# PandoRdio

![branding](http://f.cl.ly/items/2T3k0j2z2R3G2j1D2t3N/pandordio.png)

This is a Google Chrome Extension that will add a song to your Rdio
Collection when you "Thumbs Up" on Pandora.

## Installation

[The Chrome Web Store][1]
